Descripción

Interfaz estilo SCADA para control y monitoreo de PLC SIMENS mediante comunicación GET/PUT.

Reporte PDF Descargar Proyecto TIA-PORTAL tipo ARCHIVADO Descargar Proyecto TIA-PORTAL tipo ZIP

Objetivo general

• Construir un sistema tipo SCADA para control de PLC via internet, mediante una red de 3 PLC Simens (2 s7-1200 y 1 s7-1500); que pueda enviar y recibir información de los procesos que está llevando acabo el sistema.

Objetivos específicos

  • • Diseñar y construir tres circuitos con sus sensores y actuadores, que simulen un proceso.
  • • Diseñar y construir tres programas de PLC para el control de cada uno de los procesos simulando un ambiente industrial con IoT.
  • • Conectar los PLC para formar una red intercomunicada para control por botónes, HMI o una interfaz externa.
  • • Diseñar una interfaz web para el control y monitoreo de los procesos.
  • • Implementar una API en python para la comunicación con los PLC.

Conclusiones

  • • El sistema resultó tener que funcionar con una configuración GET/PUT para control individual de los PLC .
  • • La interfaz web se pudo visualizar mediante un tunel TCP expuesto con NGROK, lo que permitió la conexión desde cualquier red.
  • • Los procesos en el PLC se optimizaron para mejorar la eficiencia del sistema.
  • • Se verificó que todas las conexiones electricas tuvieran sus sistemas de protección.
